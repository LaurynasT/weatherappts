# weatherappts


# Setup
Clone the repository: `git clone https://github.com/LaurynasT/weatherapp.git`

`cd weatherapp`

# Install dependencies:

`npm install`

# Configuration

Obtain API key from OpenWeatherMap.org and set up your API key in `.env` file

# Build the project

`npm run build`

# Run HTTP-server

`http-server dist`
